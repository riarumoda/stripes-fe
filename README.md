# Stripes
A SmartNote applications making your note-taking easier.

# Important
You must set your JDK's Home on `android/gradle.properties`:
```
org.gradle.java.home=/usr/lib/jvm/java-21-openjdk
```

# Development Guide
1. Run Flutter Pub Get
```
flutter pub get
```

2. Run Project
```
flutter run --debug
```
